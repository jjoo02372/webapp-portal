개인정보 처리방침

시행일: 2026년 6월 27일  |  최종 변경일: 2026년 6월 27일

이 포털은 교사의 수업 준비를 위한 도구입니다. 학생 개인정보(이름, 학번, 성적, 활동 기록 등)를 수집하거나 저장하지 않습니다. 로그인 기능이 없으며 이용자 식별 정보를 처리하지 않습니다.

제1조 (개인정보 처리 목적)

학습 웹앱 포털(이하 "포털")은 교사가 수업 준비 목적으로 학습용 웹앱 링크를 등록·관리하기 위한 도구로, 개인정보를 수집·처리하지 않습니다. 포털 자체는 어떠한 개인정보도 서버에 전송하지 않으며, 앱 목록은 사용 중인 기기의 브라우저 로컬 스토리지에만 저장됩니다.

제2조 (처리하는 개인정보 항목)

항목처리 여부비고
학생 이름·학번미처리수집·저장하지 않음
학생 성적·활동 기록미처리수집·저장하지 않음
교사 계정·로그인 정보미처리로그인 기능 없음
IP 주소·접속 로그미처리서버 없는 정적 페이지
앱 목록 데이터로컬 저장브라우저 로컬 스토리지에만 저장, 서버 미전송

제3조 (개인정보의 보유 및 이용 기간)

서버에 전송·보관되는 개인정보가 없으므로 별도의 보유 기간이 없습니다. 앱 목록 데이터는 교사가 직접 브라우저 설정에서 로컬 스토리지를 초기화하거나 포털의 "전체 초기화" 기능을 사용하여 언제든지 삭제할 수 있습니다.

제4조 (개인정보의 파기 절차 및 방법)

로컬 스토리지에 저장된 앱 목록 데이터는 다음 방법으로 삭제할 수 있습니다.

  • 포털 "앱 관리" 화면 → "전체 초기화" 버튼 클릭
  • 브라우저 설정 → 사이트 데이터 삭제 → 해당 사이트 선택 후 삭제

제5조 (개인정보의 제3자 제공)

수집·처리하는 개인정보가 없으므로 제3자에게 제공하는 정보가 없습니다. 단, 등록된 외부 앱 URL을 클릭하면 해당 외부 서비스의 개인정보 처리방침이 적용됩니다. 외부 앱 이용 전 해당 서비스의 처리방침을 확인하시기 바랍니다.

제6조 (개인정보 처리업무의 위탁)

개인정보 처리업무를 위탁하는 업체가 없습니다.

제7조 (안전성 확보 조치)

  • 서버 없는(Serverless) 정적 페이지로 운영하여 서버 해킹 위험 없음
  • 앱 목록은 해당 기기의 브라우저에만 저장되어 외부 유출 경로 없음
  • 개인정보를 수집하지 않아 데이터 침해 위험 없음

제8조 (정보주체와 법정대리인의 권리·의무 및 행사 방법)

개인정보를 수집하지 않으므로 정보주체의 열람·정정·삭제·처리정지 요청 대상이 없습니다. 외부 앱 서비스에 대한 권리 행사는 해당 서비스 운영자에게 직접 문의하시기 바랍니다.
